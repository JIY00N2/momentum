### 4. 회고 / 느낀점

1. 오직 바닐라JS로만 구현된 웹 사이트의 CSS를 처음부터 만지려다 보니 어려웠다.
2. HTML class, id 이름 짓는것이 어려웠다.
3. 웹 사이트 디자인을 구상하는 것도 쉬운일이 아님을 느꼈다.
4. 처음으로 HTML, CSS, JS를 이용한 웹 사이트를 완성해봄으로써 성취감을 얻었다.
